# 🛍️ Walmart Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project showcases an interactive **Walmart Sales Analysis Dashboard** built using **Power BI**.
The dashboard provides insights into sales, profit, and quantity across different years, categories, and states to support data-driven decision-making.

---

## 🎯 Objective

The main objective of this project is to:

* Analyze overall sales and profit trends
* Identify high-performing product categories
* Understand regional sales performance
* Track quantity distribution across different segments

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Preparation)

---

## 📊 Key KPIs

* **Total Profit:** 108.42K
* **Total Quantity Sold:** 12K
* **Total Sales:** 725.46K

---

## 📈 Dashboard Insights

### 🔹 Sales & Profit Trend

* Sales and profit show consistent growth from **2011 to 2014**
* Peak performance observed in **2014**
* Drop observed in 2015, indicating possible business decline or incomplete data

### 🔹 Category Analysis

* Categories like **Chairs, Phones, and Storage** generate high sales
* Some categories show **low or negative profit**, indicating cost inefficiencies

### 🔹 Quantity Distribution

* High quantity sales observed in categories with moderate profit
* Helps identify high-demand but low-margin products

### 🔹 Regional Performance

* Certain states contribute more significantly to overall sales
* Geographic analysis helps identify **target markets**
